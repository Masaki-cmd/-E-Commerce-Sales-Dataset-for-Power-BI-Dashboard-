# -E-Commerce-Sales-Dataset-for-Power-BI-Dashboard-
# README.md: 利益率向上に向けたECデータ分析

## 1. プロジェクト概要

本プロジェクトは、提供されたEC販売データを用いて、企業の**利益率向上**というKGI (Key Goal Indicator) を達成するための要因分析と具体的な施策提案を行うことを目的としています。

## 2. KGIとKPIツリー

### KGI: 利益率の向上 (Improve Profit Margin)

### 提案されたKPI:
1.  **総利益 (Total Profit)**: `profit_per_order` 列の合計。
2.  **総売上高 (Total Sales)**: `sales_per_order` 列の合計。
3.  **平均注文単価 (Average Order Value - AOV)**: 総売上高 / (ユニークな`order_id`の数)。
4.  **注文数 (Number of Orders)**: ユニークな`order_id`の数。
5.  **平均割引率 (Average Discount Rate)**: `order_item_discount` / `sales_per_order` の平均。
6.  **利益率 (Profit Margin)**: (総利益 / 総売上高) * 100。

### KPIツリーの概要:
KGI「利益率の向上」は、**利益率**によって測定され、これは**総利益**と**総売上高**に分解されます。
-   **総利益**は、個々の注文アイテムからの平均利益と注文アイテム数に依存し、特に**平均割引率**が間接的に影響を与えます。
-   **総売上高**は、**平均注文単価 (AOV)** と**注文数**によって構成されます。

このツリーは、利益率を向上させるための具体的なレバー（総利益の増加、総売上高の増加、平均注文単価の向上、注文数の増加、割引率の最適化）を視覚的に示します。

## 3. 分析対象と疑問点

KGI達成のために、以下の主要な分析領域と疑問点に焦点を当てて分析を進めました。
1.  **製品カテゴリ別の利益率分析**: どの製品カテゴリが最も高い/低い利益率を達成しているか？その原因は？
2.  **顧客セグメント別の利益率分析**: どの顧客セグメントが会社全体の利益率に最も貢献しているか？特定のセグメントで利益率が低いのはなぜか？
3.  **割引戦略が利益率に与える影響**: 割引が個々の注文や全体的な利益率にどのように影響しているか？最適な割引戦略は何か？
4.  **地域別の利益率分析**: どの地域で利益率が高い/低い傾向にあるか？地域特有の戦略は？
5.  **配送ステータスと利益率の関係**: 配送に関する問題が利益を減少させているか？

## 4. データクリーニング概要

データクリーニングフェーズでは、以下の重要な処理を実施しました。
-   **欠損値の再確認**: データセット全体に欠損値がないことを確認。
-   **異常値の特定と処理方針の決定**: `order_item_discount`の最大値や`profit_per_order`の最小値、および利益がマイナスとなっている全ての注文を、ビジネス上の重要な洞察源として除外せずに保持する方針を採択。
-   **データ型の変換**: `order_date`および`ship_date`列をdatetime型に変換。

これにより、データ品質スコアは**95/100**と評価され、データは詳細な分析に適した状態にあると判断されました。

## 5. 主要な発見 (Key Findings)

### 5.1 高売上レンジにおける割引影響の深掘り
-   「1000+」売上高レンジにおいて、**高い割引率を適用しつつも高い利益率を維持している注文（128件）が存在**。
-   これは、特定の高価格商品において、戦略的な割引が顧客エンゲージメントを高めつつ、収益性を損なわない可能性があることを示唆。

### 5.2 返品された注文の調査（「配送キャンセル」として分析）
-   「Returned」ステータスの注文は存在せず、代わりに「Shipping canceled」を調査対象とした。
-   「配送キャンセル」された注文は全注文の4.32%を占めるが、これらは**総額105,591.96ドルの利益を生み出していた**。特に「Office Supplies」カテゴリがキャンセルからの利益の大部分を占め、キャンセルが発生しても再販可能であったり、キャンセル費用が利益を上回らないビジネスモデルの存在を示唆。

### 5.3 配送タイプ別収益性の分析
-   **「First Class」の平均利益率が最も高く (11.55%)**、次いで「Standard Class」 (11.42%)。一方、「Same Day」 (10.47%) と「Second Class」 (10.83%) は利益率が低い傾向。
-   迅速な配送オプションは高コストを伴うものの、「First Class」のように高い利益率を維持できるケースがある一方で、「Same Day」のように利益を圧迫しているケースがあることが判明。

### 5.4 顧客セグメント別収益性の分析
-   **「Corporate」セグメントの平均利益率が最も高く (11.42%)**、次いで「Consumer」 (11.25%)、最も低いのが「Home Office」 (11.08%)。
-   Corporateセグメントは健全な利益構造を持つ一方、Home Officeセグメントは価格に敏感か、提供サービスのコストが高い可能性。

### 5.5 利益がマイナスとなっている注文の傾向分析
-   全注文の**18.70%が利益を伴わないか損失**を計上しており、これは無視できない割合である。
-   損失額が大きい注文は、「Furniture」や「Technology」カテゴリの高額商品に多く、高額な売上にもかかわらず、大きな割引や高コスト構造により損失が発生している傾向が見られた。
-   特定の顧客セグメント（Corporate、Consumer）や配送タイプ（First Class、Same Day）で損失が偏る傾向も確認された。

## 6. ビジネスへの提言 (Business Recommendations)

上記の発見に基づき、利益率向上のための以下の具体的な提言を行います。

1.  **高利益・高割引戦略の最適化と横展開**:
    -   「1000+」売上高レンジで高割引・高利益を両立している注文の特性（製品、顧客、割引のタイミング・種類など）をさらに深掘りし、その成功要因を特定。特定された成功パターンを他の製品やプロモーション戦略に応用し、収益性を高める。

2.  **配送タイプ戦略の見直し**:
    -   **「First Class」**が最も高い利益率を示している点に注目し、その強みをさらに活用する戦略を検討。
    -   **「Same Day」**や**「Second Class」**など、利益率が低い配送タイプについて、コスト構造を詳細に分析し、配送パートナーとの再交渉、物流プロセスの効率化、または顧客への請求価格の適正化を検討。特定の製品や顧客に限定して提供することも有効。

3.  **顧客セグメントに合わせた戦略の策定**:
    -   **「Corporate」セグメント**の利益率が高い理由を深掘りし、その成功要因を他のセグメントへのアプローチに生かす。
    -   **「Home Office」セグメント**の利益率向上に向け、価格競争力の高い製品の提供、コスト効率の良いプロモーション、または高付加価値サービスの提案を検討し、単価向上を目指す。

4.  **損失発生注文の集中改善**:
    -   損失を計上している約18.70%の注文、特に「Furniture」や「Technology」カテゴリの高額商品における高割引による損失は緊急の課題。製品の原価、割引の適用ルール、返品・キャンセルポリシー、物流コストのすべてを横断的に見直し、損失の発生メカニズムを特定。
    -   返品された（またはキャンセルされた）注文のデータから、具体的な返品・キャンセル理由を深掘りし、製品品質の改善、商品説明の正確性の向上、または配送の信頼性向上に繋げる。

## 7. 今後の分析課題

さらなる深掘りのために以下の分析課題が残されています。
-   **高利益・高割引注文の具体的原因究明**: 特定された高割引・高利益注文の製品レベル、顧客レベルでの詳細な背景を特定し、再現性のある成功戦略を構築。
-   **損失注文のコストドライバー分析**: `profit_per_order`がマイナスとなる主要な要因を定量的に分解し、最も影響の大きいドライバーを特定することで、最も効果的な改善策を打つ。
-   **顧客生涯価値 (CLV) と利益率の関連**: 顧客セグメントごとの利益率だけでなく、顧客生涯価値も考慮に入れ、短期的な利益と長期的な顧客関係のバランスを最適化する戦略を構築。

これらの提言と今後の課題を通じて、データに基づいた意思決定を支援し、KGIである「利益率の向上」に貢献することを目指します。
